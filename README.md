# 📊 Data App - Upload e Análise de Dados

Aplicação web desenvolvida com Streamlit + Supabase para ingestão e visualização de dados a partir de arquivos CSV.

---

## 🚀 Funcionalidades

🔐 Autenticação de usuários (login/cadastro)  
📁 Upload de arquivos CSV  
👀 Pré-visualização dos dados enviados  
🗄️ Integração com banco de dados (Supabase)  
📊 Estrutura pronta para análise de dados  

---

## 🛠️ Tecnologias utilizadas

- Python
- Streamlit
- Supabase
- Pandas
- SQL

## 📊 Fluxo do projeto

1. Usuário realiza login/cadastro  
2. Envia um arquivo CSV  
3. Sistema valida e exibe pré-visualização  
4. Dados podem ser enviados para o banco  
5. Estrutura pronta para análise em dashboards  

---

## 📈 Dashboard (Looker Studio)

Os dados podem ser utilizados para criação de dashboards com métricas como:

- Receita total
- Vendas por produto
- Vendas por cidade
- Evolução temporal
